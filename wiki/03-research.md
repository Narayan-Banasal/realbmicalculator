## Executive Summary (from research)
"bmi calculator" is high-volume, high-competition. Dominated by:
- Authority (CDC, NHLBI/NIH)
- Calculator networks (calculator.net — deepest content, ad-heavy; thecalculatorsite)
- Modern tools (Omni Calculator — best UX with live scale/gauge; bmi-calculator.net — good hub + formula toggle; smartbmicalculator — advanced with ethnicity etc.)

**Win condition (Compile Future playbook)**: Differentiated **premium UX** (instant + dark + visuals + mobile) + **sharper brand/domain** + **deeper honest content** + FAQ schema + visible trust signals + US CPM focus. Not "another plain form".

Design benchmark: Omni + bmi-calculator.net  
Content benchmark: calculator.net  
Trust: CDC/NIH disclaimers (without their dated UI)

## Competitor Feature Matrix (key rows)
- Live result / no submit: only Omni really strong → **we do this + more**
- Color scale / gauge: Omni best → **we have animated marker + full spectrum + synced body guides**
- Dark mode: rare → **we have excellent one**
- Mobile-first + accessibility: mediocre on many → **primary for us**
- FAQ + schema: weak on some → **rich + JSON-LD**
- Share result URL: only Omni → **we have + copy button**
- Ads clutter: heavy on calculator.net → **light, below fold**
- Related calculators / hub: excellent on bmi-calculator.net → future opportunity (WHR, ideal weight, etc.)

## UX Gaps We Exploited (directly from research list)
1. Hero calculator above fold (no scrolling to use the tool).
2. (Adapted) clean health-tech aesthetic instead of generic.
3. Animated BMI gauge + marker.
4. Result card with large number + category + healthy weight range + BMI Prime.
5. One-tap unit switch (sticky, persisted).
6. **Dark mode** (must-have, missing from most).
7. Copy result / share link (viral).
8. Honest limitations + "for athletes / pregnancy / seniors" context in copy.
9. Illustration system (original SVGs per category — not stock).
10. 3D explorer + gallery as extra delight layer.

## Visual Direction Applied
"Clinical clarity": DM Sans + Fraunces, emerald accent for health, off-white canvas, clean cards, generous whitespace between bands, tight interior content. No sixth accent color. Sentence case + negative tracking on display headlines. Mono only for labels.
