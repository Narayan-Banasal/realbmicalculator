# Real BMI Calculator

Free BMI (body mass index) calculator for **realbmicalculator.com**.

Built with the [Compile Future micro-tool playbook](https://www.youtube.com/watch?v=tDeLeyPvZn0): Astro, Tailwind v4, Cloudflare, SEO-first MPA.

## Commands

```bash
npm install
npm run dev      # http://localhost:4321
npm run build
npm run deploy   # build + wrangler (after wrangler login)
```
